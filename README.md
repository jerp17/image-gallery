# image-gallery
grid image-gallery
Created a grid-based image gallery using HTML and CSS.
Added click event listeners to each image thumbnail using JavaScript.
Created a hidden modal element that will display the enlarged image.
Dynamically update modal content based on the clicked image.
Implemented open and close functionality using JavaScript.
Added background overlay to focus user attention on modal.
Prevented background scrolling when modal is open.
Added keyboard support (ESC key) to close modal.
Tested accessibility and responsiveness thoroughly.
